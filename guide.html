MusicBox User Guide

Using the MusicBox Application

Everything you need to know about the MusicBox desktop app

Getting Started

1

Download & Run

Download the MusicBox application for your operating system from the homepage. Simply double-click the downloaded file to run it - no installation needed!

Windows: Double-click MusicBox.exe
Linux: Right-click the file, select Properties, check "Allow executing file as program", then double-click it

2

Connect Your Device

Plug your MusicBox device into your computer using a USB cable. The application will automatically find and connect to it.

If it doesn't connect automatically, click the "Auto-Detect" button in the Config tab.

3

Start Playing Music

Play music in any supported application (Spotify, Chrome, VLC, etc.) and your MusicBox device will automatically display what's playing!

Using the Application

Status Tab

The Status tab displays your currently playing music

The Status tab shows you what's currently playing:

  • Track Title: The name of the song
  • Artist: Who performs the song
  • Time: Current position and total duration
  • Status: Playing, Paused, or Stopped

Config Tab

Configure your device and application settings

The Config tab lets you change settings:

  • Change Port: Manually select which USB port your device is connected to
  • Auto-Detect: Automatically find your MusicBox device
  • Launch at Startup: Check this box to make MusicBox start automatically when your computer starts

Debug Tab

Debug information and troubleshooting tools

The Debug tab helps with troubleshooting:

  • System Information: Shows app version and memory usage
  • Connection Debug: Shows connection status messages
  • Restart Service: Click this button to restart the connection if something goes wrong
  • View Logs: Opens the folder containing detailed log files

Need Help?

If Something Isn't Working

  1. Try clicking the "Restart Service" button in the Debug tab
  2. Make sure your device is properly plugged in via USB
  3. Try clicking "Auto-Detect" in the Config tab
  4. If the problem persists, click "View Logs" in the Debug tab to find your log files

Contact Support

If you need additional help, please send us:

  • A description of the problem
  • Your log files (found by clicking "View Logs" in the Debug tab)

Send to: support@musicbox.example.com

Or submit a support request via our Google Form

Setting Up Your MusicBox Device

Hardware connection and setup

What You Need

✓ MusicBox Device

Your assembled MusicBox device with ESP32-S3 and display

✓ USB Cable

A USB cable to connect your device to your computer (must support data transfer, not just charging)

✓ Computer

Windows PC or Linux computer with a USB port

Setup Steps

1

Connect the Device

Plug your MusicBox device into your computer using the USB cable. The device should power on and the display should light up.

2

Launch the Application

Run the MusicBox application on your computer. It will automatically detect your device.

3

Verify Connection

Check the Config tab to see if your device port is showing. If you see a port name (like "COM3" or "/dev/ttyUSB0"), you're connected!

4

Test It Out

Play some music in Spotify, Chrome, or any other supported app. Your device should start displaying the track information!

Common Issues

Device Won't Connect

  • Make sure the USB cable supports data transfer (some cables are charging-only)
  • Try a different USB port on your computer
  • Unplug the device, wait 5 seconds, and plug it back in
  • Click "Auto-Detect" in the Config tab

Display Not Showing Music

  • Make sure music is actually playing in a supported app
  • Check that the device is connected (see Config tab)
  • Try clicking "Restart Service" in the Debug tab

Still Having Problems?

Click "View Logs" in the Debug tab and send the log files to our support team at support@musicbox.example.com
